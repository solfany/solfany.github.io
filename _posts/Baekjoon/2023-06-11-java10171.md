## 문제

아래 예제와 같이 고양이를 출력하시오.

## 입력

없음.

## 출력

고양이를 출력한다.

## 예제 입력 1

```

```

## 예제 출력 1

```
\    /\
 )  ( ')
(  /  )
 \(__)|
```

```java
package joon01;

public class N10171 {

	public static void main(String[] args) {
		System.out.println("\\    /\\");
		System.out.println(" )  ( ')");
		System.out.println("(  /  )");
		System.out.println(" \\(__)|");
	}
}
```

> 한줄에 줄 바꿈을 사용해 코드를 작성했더니 출력 양식이 틀렸다고 나온다.
> 한줄씩 출력해야 하나보다..
