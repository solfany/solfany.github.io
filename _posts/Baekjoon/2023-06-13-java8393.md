# 합 다국어

## 문제

n이 주어졌을 때, 1부터 n까지 합을 구하는 프로그램을 작성하시오.

## 입력

첫째 줄에 n (1 ≤ n ≤ 10,000)이 주어진다.

## 출력

1부터 n까지 합을 출력한다.

## 예제 입력 1

```
3

```

## 예제 출력 1

```
6
```

```java
package backjoon;

import java.util.Scanner;

public class N8393 {
		public static void main(String[] args) {
			Scanner s = new Scanner(System.in);

			int a = s.nextInt();
			s.close();
			int sum = 0;

			for (int i = 1; i <= a; i++) {
				sum += i;
			}
			System.out.println(sum);
		}
	}
```
